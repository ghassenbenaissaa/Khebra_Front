## 🔎 Overview
**Khebra_Front** is a frontend project built with **Angular 18**.  
It provides a modern and responsive user interface for the Khebra platform, connecting seamlessly with the backend services (`Khebra_Back`).  

---

## ✅ Prerequisites
Before installing, make sure you have:
- [Node.js 20+](https://nodejs.org/)  
- [Angular CLI 18](https://angular.dev/tools/cli)  
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)  

---

## ⚡ Installation  

### 1. Clone the repository  
```sh
git clone https://github.com/ghassenbenaissaa/Khebra_Front
cd Khebra_Front
```

### 2. Install dependencies  
```sh
npm install
```
or  
```sh
yarn install
```

---

## ▶️ Run the Application  

Start the development server:  
```sh
ng serve
```

Then open your browser at:  
👉 [http://localhost:4200](http://localhost:4200)  

---

## 🛠️ Build for Production  
```sh
ng build --configuration production
```

The build output will be stored in the `dist/` directory.  

---

## 🧪 Testing  

Run unit tests:  
```sh
ng test
```

Run end-to-end tests:  
```sh
ng e2e
```
